# Tutorial LibUV

LibUV es una biblioteca para programación orientada a eventos en C, utilizada principalmente como el núcleo del motor de javascript **v8**

## Instalación

~~~
sudo apt install libuv1-dev
~~~

## Compilación

~~~
mkdir build
cd build
cmake ../src
~~~
